# Setting Up Damira AI

Damira AI is a free AI assistant for network engineers and IT students. It's pre-installed in these Codespaces to help you study for CompTIA certifications.

## Quick Setup (2 minutes)

### 1. Get Your Free API Key
- Go to [damiraai.com](https://damiraai.com)
- Click **Sign Up** (no credit card required)
- Copy your API key from the dashboard

### 2. Configure the Extension
- Open VS Code Settings (`Ctrl+,` or `Cmd+,`)
- Search for **"Damira"**
- Paste your API key in the **API Key** field

### 3. Start Studying with Damira
- Open the Damira chat panel (click the Damira icon in the sidebar)
- Ask about any concept, paste lab output, or describe what you're stuck on

## What Can Damira Help With?

- **Exam concepts**: "Explain the difference between TCP and UDP for Network+"
- **Lab troubleshooting**: "My static route isn't working, here's my routing table..."
- **Practice questions**: "Give me 5 Network+ practice questions about subnetting"
- **Command explanations**: "What does 'ip route add default via 10.1.1.1' do?"
